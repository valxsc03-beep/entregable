# Ebooks Gratuitos — Semana 04: Strings y Template Literals

## 📚 Libros recomendados

### Eloquent JavaScript (Disponible online gratis)

- **URL**: https://eloquentjavascript.net/
- **Capítulo relevante**: Chapter 1 — Values, Types, and Operators (sección Strings)
- **Idioma**: Inglés (con traducción al español disponible: https://eloquentjavascript.net/Eloquent_JavaScript_small.pdf)
- **Descripción**: Introduce los strings como tipo primitivo, caracteres de escape y cómo JavaScript representa texto internamente

### JavaScript for Impatient Programmers

- **URL**: https://exploringjs.com/impatient-js/
- **Capítulo relevante**: Chapter 19 — Unicode — a brief introduction; Chapter 20 — Strings
- **Idioma**: Inglés
- **Descripción**: Referencia moderna con cobertura completa de métodos de string ES2023, template literals y manejo de Unicode

### You Don't Know JS: Types & Grammar

- **URL**: https://github.com/getify/You-Dont-Know-JS/blob/1st-ed/types%20%26%20grammar/README.md
- **Capítulo relevante**: Chapter 2 — Values (sección Strings)
- **Idioma**: Inglés
- **Descripción**: Explica por qué los strings son inmutables, cómo se compartan comportamientos con arrays y las diferencias clave entre ambos tipos
