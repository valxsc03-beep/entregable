# Rúbrica de Evaluación — Semana 04: Strings y Template Literals

## 📊 Tabla de criterios

| #   | Criterio                            | Indicadores                                                                                     | Puntos      |
| --- | ----------------------------------- | ----------------------------------------------------------------------------------------------- | ----------- |
| 1   | **El script funciona sin errores**  | Se ejecuta con `node script.js` sin errores ni excepciones                                      | 20 pts      |
| 2   | **Uso de métodos de string**        | Usa al menos 5 métodos diferentes (`trim`, `toUpperCase`, `includes`, `slice`, `replace`, etc.) | 25 pts      |
| 3   | **Template literals**               | Usa template literals con interpolación en lugar de concatenación con `+`                       | 20 pts      |
| 4   | **Extracción y transformación**     | Aplica `slice`, `split` o `indexOf` para manipular texto del dominio                            | 15 pts      |
| 5   | **Adaptación coherente al dominio** | Los strings y mensajes son consistentes con el dominio asignado                                 | 10 pts      |
| 6   | **Nomenclatura y código limpio**    | Nombres en camelCase, sin `var`, sin concatenación con `+` donde hay template literals          | 10 pts      |
|     | **Total**                           |                                                                                                 | **100 pts** |

**Mínimo para aprobar**: 70 pts

## ✏️ Nota adicional

Se descontarán **5 puntos** por cada uno de los siguientes incumplimientos:

- Uso de `var` en lugar de `const`/`let`
- Uso de concatenación con `+` donde corresponde template literal
- Variables sin nombre descriptivo (`x`, `a`, `temp`, etc.)
- Código copiado de otro aprendiz (nota 0 y llamado de atención)
