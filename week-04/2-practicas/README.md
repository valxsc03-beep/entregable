# Prácticas — Semana 04: Strings y Template Literals

Cinco ejercicios guiados de dificultad creciente. Cada ejercicio tiene un `starter/` para descomentar y un `solution/` de referencia.

| #   | Ejercicio                                                 | Concepto central                                |
| --- | --------------------------------------------------------- | ----------------------------------------------- |
| 1   | [Propiedades de strings](ejercicio-01-propiedades/)       | `.length`, acceso por índice                    |
| 2   | [Métodos de transformación](ejercicio-02-transformacion/) | `toUpperCase`, `trim`, `replace`, `repeat`      |
| 3   | [Métodos de búsqueda](ejercicio-03-busqueda/)             | `includes`, `startsWith`, `endsWith`, `indexOf` |
| 4   | [Extracción y manipulación](ejercicio-04-extraccion/)     | `slice`, `split`, encadenamiento                |
| 5   | [Template literals](ejercicio-05-template-literals/)      | interpolación, multilínea, expresiones          |

---

## Instrucciones generales

1. Abre el `README.md` del ejercicio y lee el paso a paso
2. Abre `starter/index.js` y descomenta el código de cada paso
3. Ejecuta con `node starter/index.js` y verifica la salida
4. Consulta `solution/index.js` solo si te bloqueas

> **No copies la solution directamente** — el aprendizaje está en ejecutar cada paso y observar el resultado.
