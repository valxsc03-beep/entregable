# Semana 04: Strings — Texto y Template Literals

> **Etapa 0 — Fundamentos de Programación** | Semana 4 de 10

---

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Trabajar con strings usando sus propiedades y métodos principales
- Usar template literals para construir textos dinámicos e interpolados
- Manejar secuencias de escape dentro de strings
- Aplicar métodos de búsqueda, transformación y extracción sobre texto
- Reemplazar la concatenación con `+` por template literals (ES2023 style)

---

## 📚 Requisitos previos

- Semana 01: `console.log()`, tipos primitivos
- Semana 02: `const`/`let`, tipos y conversiones
- Semana 03: operadores y expresiones
