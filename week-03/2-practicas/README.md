# Prácticas — Semana 03: Operadores y Expresiones

> ⚠️ **Importante**: Este material es de **aprendizaje guiado — NO se entrega**.
> El único entregable obligatorio de la semana es el **proyecto** en `3-proyecto/`.

## 🎯 Objetivo

Practicar todos los operadores de JavaScript a través de ejercicios progresivos antes de aplicarlos en el proyecto integrador.

## 🚀 Cómo ejecutar un ejercicio

```bash
# Ejecutar el starter (tu trabajo)
node ejercicio-01-aritmeticos/starter/index.js

# Ver la solución cuando termines
node ejercicio-01-aritmeticos/solution/index.js
```

## 📌 Metodología

Cada ejercicio sigue el mismo formato:

1. **README.md** — Explica los conceptos y los pasos
2. **starter/index.js** — Código comentado que debes descomentar paso a paso
3. **solution/index.js** — Solución completa para comparar al finalizar

**Recomendación**: No mires la solución antes de intentar cada paso por tu cuenta.
