# Semana 03: Operadores y Expresiones

> 🟪 **Etapa 0 — Fundamentos de Programación** | Semana 3 de 10

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Usar todos los operadores aritméticos, incluido el de potencia `**`
- Aplicar operadores de asignación compuesta (`+=`, `-=`, `*=`, `/=`)
- Comparar valores usando `===` y `!==` (igualdad estricta)
- Entender por qué `==` es peligroso y `===` es la forma correcta
- Combinar condiciones con operadores lógicos `&&`, `||`, `!`
- Leer y respetar el orden de precedencia de operadores
- Identificar cuándo ocurre coerción implícita y cómo evitarla
