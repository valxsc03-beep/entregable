# 📖 Glosario — Semana 1

Términos clave de la semana ordenados alfabéticamente.

---

## A

### Algoritmo
Secuencia finita de instrucciones ordenadas que resuelven un problema. Todo programa es un algoritmo expresado en un lenguaje de programación.

---

## B

### Boolean
Tipo de dato con solo dos valores posibles: `true` (verdadero) o `false` (falso). Es el tipo de dato de las decisiones lógicas.

```javascript
console.log(true);   // true
console.log(false);  // false
```

---

## C

### Ciclo Entrada → Proceso → Salida (E→P→S)
Patrón fundamental de todo programa: recibe datos de entrada, los procesa mediante instrucciones, y produce una salida (resultado).

### Comentario
Texto en el código que JavaScript ignora completamente. Sirve para explicar el propósito del código. Se escriben en español en este bootcamp.

```javascript
// Comentario de una línea
/* Comentario
   multilínea */
```

### console.log()
Instrucción que muestra un valor o mensaje en la consola. Es la herramienta básica de comunicación y depuración en JavaScript.

```javascript
console.log('Hola, mundo');  // Hola, mundo
```

---

## D

### Declaración (sentencia)
Instrucción completa que realiza una acción. Diferente de una expresión: una declaración hace algo, una expresión produce un valor.

---

## E

### Expresión
Cualquier trozo de código que produce un valor. JavaScript la evalúa antes de usarla.

```javascript
10 + 5      // produce 15
'Hola'      // produce "Hola"
3 > 2       // produce true
```

---

## I

### Instrucción
Una orden individual dentro de un programa. Cada `console.log()` es una instrucción.

---

## J

### JavaScript
Lenguaje de programación creado en 1995. El único lenguaje que entienden todos los navegadores web. También corre en servidores con Node.js.

---

## L

### Lenguaje de programación
Intermediario que permite a los humanos escribir instrucciones en un formato comprensible, que se traduce automáticamente a código máquina.

### Literal (valor literal)
Un dato escrito directamente en el código, sin cálculo previo.

```javascript
'Hola'   // string literal
42       // number literal
true     // boolean literal
```

---

## N

### NaN (Not a Number)
Valor especial de tipo number que representa un resultado matemático inválido.

```javascript
console.log('texto' * 2);   // NaN
```

### Node.js
Entorno de ejecución que permite correr JavaScript fuera del navegador, directamente desde la terminal.

### Number
Tipo de dato para representar números, tanto enteros como decimales. El separador decimal es el punto `.`.

```javascript
42       // entero
3.14     // decimal
-10      // negativo
```

---

## P

### Programa
Secuencia de instrucciones ordenadas que una computadora ejecuta para resolver un problema o realizar una tarea.

---

## S

### String
Tipo de dato para representar texto. Se escribe entre comillas simples, dobles o backticks.

```javascript
'Hola, mundo'
"También válido"
```

---

## T

### Terminal
Interfaz de texto para dar instrucciones al sistema operativo. También llamada consola o línea de comandos.

### typeof
Operador que devuelve el tipo de un valor como string.

```javascript
typeof 'hola'   // "string"
typeof 42       // "number"
typeof true     // "boolean"
```

### Tipo de dato
Clasificación de un valor que determina qué operaciones se pueden hacer con él.

---

## V

### VS Code (Visual Studio Code)
Editor de código gratuito de Microsoft. El más popular para desarrollo JavaScript.
