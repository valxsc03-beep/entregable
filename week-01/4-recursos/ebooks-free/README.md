## 📚 Cómo Usar Estos Recursos

### Para la Semana 01

**Lectura Prioritaria:**
1. **JavaScript.info** - Secciones de variables, functions
2. **Eloquent JavaScript** - Capítulos 1-4
3. **Exploring ES2023** - Capítulos de let/const, arrows, destructuring

**Lectura Complementaria:**
- MDN Guide como referencia
- YDKJS ES2023 & Beyond para profundizar

### Estrategia de Lectura

1. **Lee activamente**: Reproduce los ejemplos en tu editor
2. **Toma notas**: Escribe conceptos clave con tus palabras
3. **Practica**: Haz los ejercicios del libro
4. **Vuelve**: Relee secciones complicadas
5. **Aplica**: Usa lo aprendido en tus proyectos

### Orden Sugerido

```
Semana 01:
├── JavaScript.info (Variables y Functions)
├── Eloquent JavaScript (Capítulos 1-4)
└── Exploring ES2023 (Capítulos relevantes)

Semana 02+:
├── YDKJS (toda la serie)
├── JavaScript Allongé
└── Design Patterns
```
