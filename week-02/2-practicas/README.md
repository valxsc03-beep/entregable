## 🛠️ Cómo trabajar con los ejercicios

Cada ejercicio tiene la misma estructura:

```
ejercicio-XX-nombre/
├── README.md        ← Instrucciones y ejemplos
├── starter/
│   └── index.js    ← Tu archivo de trabajo (descomenta código)
└── solution/
    └── index.js    ← Solución de referencia
```

### Pasos para cada ejercicio

1. Lee el `README.md` del ejercicio
2. Abre `starter/index.js`
3. Descomenta el código de cada paso según las instrucciones
4. Ejecuta con `node starter/index.js`
5. Compara con `solution/index.js` cuando termines

---

## ⚠️ Formato de los ejercicios

Los ejercicios son **tutoriales guiados**. El código está comentado y tú lo activas paso a paso. Este formato:

- Te permite leer primero, ejecutar después
- Facilita identificar qué produce cada bloque de código
- Evita errores de sintaxis al comenzar

---

## ✅ Checklist de aprendizaje

Al terminar todos los ejercicios, deberías poder:

- [ ] Decidir cuándo usar `const` vs `let`
- [ ] Identificar los 5 tipos primitivos y usar `typeof`
- [ ] Distinguir `null` de `undefined`
- [ ] Convertir entre tipos con `Number()`, `String()`, `Boolean()`
- [ ] Nombrar variables con las convenciones correctas
- [ ] Escribir números grandes con `_` para mejorar legibilidad
