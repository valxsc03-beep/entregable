# Ebooks Gratuitos — Semana 05: Condicionales

## 📚 Libros recomendados

### Eloquent JavaScript (Disponible online gratis)

- **URL**: https://eloquentjavascript.net/
- **Capítulo relevante**: Chapter 2 — Program Structure (sección Conditional execution)
- **Idioma**: Inglés (con traducción al español disponible)
- **Descripción**: Introduce `if/else` y bucles como mecanismos de control de flujo con ejemplos claros para principiantes

### JavaScript for Impatient Programmers

- **URL**: https://exploringjs.com/impatient-js/
- **Capítulo relevante**: Chapter 13 — Controlling flow (if-else, switch, while, for)
- **Idioma**: Inglés
- **Descripción**: Referencia moderna con cobertura de `??` y `?.` de ES2020, incluyendo comparación con enfoques anteriores

### You Don't Know JS: Types & Grammar

- **URL**: https://github.com/getify/You-Dont-Know-JS/blob/1st-ed/types%20%26%20grammar/README.md
- **Capítulo relevante**: Chapter 4 — Coercion (sección Implicit: \* To Boolean)
- **Idioma**: Inglés
- **Descripción**: Explica en profundidad los valores truthy y falsy, cómo funciona la coerción implícita en contextos booleanos y por qué `[] == false` es `true` pero `if ([])` ejecuta el bloque
