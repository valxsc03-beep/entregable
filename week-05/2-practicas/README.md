1. Lee el `README.md` de cada ejercicio
2. Abre `starter/index.js`
3. Descomenta las líneas indicadas paso a paso
4. Ejecuta con `node starter/index.js` y observa el resultado
5. Compara con `solution/index.js` si tienes dudas
