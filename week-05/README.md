# Semana 05: Condicionales

> **Etapa 0 — Fundamentos de Programación** | Semana 5 de 10

---

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Controlar el flujo de ejecución de un programa con `if`, `else if` y `else`
- Simplificar condiciones simples con el operador ternario `? :`
- Seleccionar entre múltiples opciones con `switch`
- Aplicar `??` (nullish coalescing) y `?.` (optional chaining) de ES2020
- Distinguir valores truthy y falsy para escribir condiciones precisas
- Usar operadores lógicos `&&`, `||` como expresiones condicionales

---

- Semana 01: `console.log()`, tipos primitivos, `typeof`
- Semana 02: `const`/`let`, tipos y conversiones
