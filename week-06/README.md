# Semana 06: Bucles — Repetición

> **Etapa 0 — Fundamentos de Programación** | Semana 6 de 10

---

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Usar el bucle `for` clásico para repetir acciones un número definido de veces
- Aplicar `while` y `do...while` para repetir hasta que una condición cambie
- Iterar sobre arrays y strings con `for...of` (sintaxis moderna preferida)
- Controlar el flujo de bucles con `break` y `continue`
- Construir bucles anidados para trabajar con datos bidimensionales
- Implementar contadores y acumuladores para procesar listas de datos

---

## 📚 Requisitos previos

- Semana 01: `console.log()`, tipos primitivos
- Semana 02: `const`/`let`, conversiones de tipo
- Semana 03: operadores aritméticos (`+`, `+=`, `++`) y de comparación (`<`, `>`, `===`)
- Semana 04: template literals
- Semana 05: condicionales `if/else`, `break` dentro de `switch`
