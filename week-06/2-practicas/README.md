# Prácticas — Semana 06: Bucles

Esta semana tienes **5 ejercicios guiados**. Cada uno está diseñado para que aprendas descomentando código paso a paso.

| #   | Ejercicio                                                   | Concepto principal           | Tiempo estimado |
| --- | ----------------------------------------------------------- | ---------------------------- | --------------- |
| 1   | [ejercicio-01-for-clasico](ejercicio-01-for-clasico/)       | `for` y variantes            | ~35 min         |
| 2   | [ejercicio-02-while](ejercicio-02-while/)                   | `while` y `do...while`       | ~35 min         |
| 3   | [ejercicio-03-for-of](ejercicio-03-for-of/)                 | `for...of` en arrays/strings | ~35 min         |
| 4   | [ejercicio-04-break-continue](ejercicio-04-break-continue/) | `break` y `continue`         | ~35 min         |
| 5   | [ejercicio-05-acumuladores](ejercicio-05-acumuladores/)     | Contadores y acumuladores    | ~40 min         |

## 🚀 Cómo ejecutar cada ejercicio

```bash
node starter/index.js    # ver el estado inicial
node solution/index.js   # ver la solución completa
```

## 📌 Recuerda

- Trabaja siempre en `starter/index.js`
- Descomenta el código **sección por sección**
- Verifica la salida en consola antes de continuar
- Solo consulta `solution/index.js` si estás bloqueado
